# Overview  
起動しているターミナル、プロセスを落とすプログラムになります。　　

---

# Download  
ホームなどにスクリプトのみをダウンロードする場合は　　

`wget https://raw.githubusercontent.com/HappyTatsuhito/kill_process/master/kill_process.py`　　

でダウンロードして下さい。　　

---

# Usage  
`python kill_process.py -オプション`　　
`./kill_process.py -オプション`　　
などで実行して下さい。　　
オプションには`-a`と`-r`
